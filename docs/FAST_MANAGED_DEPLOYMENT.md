# Step 33 — Fast managed deployment path

Chosen path:

```text
Backend + Postgres: Render
Frontend: Vercel
Source control: GitHub
```

Why this path:

```text
- GitHub gives us a release checkpoint and deploy source.
- Render can host the FastAPI backend and a managed Postgres database.
- Vercel is fast for the Vite/React frontend.
- This avoids running a VPS before the MVP is validated.
```

## High-level deployment order

```text
1. Initialize Git and push to GitHub.
2. Deploy backend + Postgres on Render.
3. Copy the Render backend URL.
4. Deploy frontend on Vercel.
5. Add VITE_API_BASE_URL in Vercel.
6. Copy the Vercel frontend URL.
7. Add FRONTEND_ALLOWED_ORIGINS in Render.
8. Run post-deploy smoke check.
9. Test sign in, Feed, Forge, Evolution manually.
```

## Backend on Render

Recommended: use the included `render.yaml` Blueprint.

Render will create:

```text
jotpop-api
jotpop-db
```

Important backend env variables:

```text
DATABASE_URL              auto-wired from jotpop-db
JWT_SECRET_KEY            generated by Render
JWT_ALGORITHM             HS256
DEV_USER_EMAILS           ale@example.com
FRONTEND_ALLOWED_ORIGINS  set after frontend deployment
```

Backend health after deploy:

```text
https://YOUR_RENDER_BACKEND_URL/health
```

Expected version after Step 33:

```json
{"status":"ok","service":"jotpop-api","version":"0.26.0"}
```

## Frontend on Vercel

Project settings:

```text
Framework Preset: Vite
Root Directory: frontend
Build Command: npm run build
Output Directory: dist
```

Environment variable:

```text
VITE_API_BASE_URL=https://YOUR_RENDER_BACKEND_URL
```

No trailing slash.

## CORS final step

After Vercel deploys, go back to the Render backend service and set:

```text
FRONTEND_ALLOWED_ORIGINS=https://YOUR_VERCEL_FRONTEND_URL
```

Then redeploy/restart the Render backend.

## Post-deploy smoke check

```bash
./scripts/post_deploy_smoke_check.sh \
  https://YOUR_RENDER_BACKEND_URL \
  https://YOUR_VERCEL_FRONTEND_URL
```

Then open the Vercel URL and test:

```text
- login
- Feed swipe
- empty Micro-Jot up-skip
- written Micro-Jot right-save
- challenge to Forge
- Forge ritual
- Evolution avatar
- Dev smoke check
```
